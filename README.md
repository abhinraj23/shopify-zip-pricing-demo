# Shopify ZIP Pricing Demo

## Overview

This project demonstrates ZIP-based dynamic pricing on a Shopify product page.

Customers enter a ZIP code and click **Check Price**. The Shopify frontend sends the ZIP code to a FastAPI backend, which returns the corresponding price.

## Technologies Used

- Shopify
- JavaScript
- Python
- FastAPI
- Render

## ZIP Pricing

| ZIP Code | Price |
|----------|--------|
| 75028 | $1499 |
| 10001 | $1699 |
| 90210 | $1799 |

## How It Works

1. User enters a ZIP code.
2. Shopify sends the ZIP code to the FastAPI backend.
3. Backend returns the corresponding price.
4. Price is displayed on the product page.
